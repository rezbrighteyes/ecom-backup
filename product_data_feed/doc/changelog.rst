Changelog
=========

`18.0.1.1.6`
------------

- Add a column option to remove completely all tags from the HTML.

`18.0.1.1.5`
------------

- Add a feed option to show the full error traceback when an error occurs during feed generation.

`18.0.1.1.4`
------------

- Implement the binary fields using in feed columns. The binary/image field can also the option to transform to the image URL.

`18.0.1.1.3`
------------

- Make the Date and Datetime field types available for selection in the Model Field columns.

`18.0.1.1.2`
------------

- Add the option to format numbers to local standards, for instanse: "1000.00" -> "1.000,00".

`18.0.1.1.1`
------------

- Improve UA translations.

- Skip escaping CDATA for empty values.

- Add buttons to open a product template from a product variant form, and product variants from the template form.

`18.0.1.1.0`
------------

- Improve a visibility of the token regeneration button.

- Add a new model "product.data.feed.type" to distinguish feeds for the same recipient.

- Add methods for getting XML tag names and option to indicate the partial caching support.

`18.0.1.0.0`
------------

- Migration from 17.0.
